Lógica del servidor:
Recibe datos enviados por el cliente mediante la lógica del cliente
Los procesa
Guarda de forma "permanente"

Lógica del cliente:
Se comunica para intercambio de datos con la lógica del servidor
Muestra los datos del usuario mediante interfaz web
Entiende los controles incluidos en la interfaz


Aplicaciones de una página (SPA)
